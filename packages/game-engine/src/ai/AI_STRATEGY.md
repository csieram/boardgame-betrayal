# AI Traitor 策略文檔

## 概述

本文檔描述 Betrayal 桌遊單人模式中叛徒 AI 的決策策略。

## 檔案結構

```
packages/game-engine/src/ai/
├── index.ts              # AI 模組入口
├── AIDecisionEngine.ts   # AI 決策引擎
├── TraitorAI.ts          # 叛徒 AI 控制器
└── __tests__/
    └── TraitorAI.test.ts # 測試檔案
```

## 決策策略摘要

### 1. 決策流程

```
遊戲狀態 → 合法行動生成 → 決策評分 → 難度調整 → 行動選擇
```

### 2. 合法行動介面

AI 只能使用以下合法行動：

- **移動 (move)**: 移動到相鄰的已發現房間
- **攻擊 (attack)**: 攻擊同一房間的敵對玩家
- **使用物品 (useItem)**: 使用持有的物品卡或預兆卡
- **探索 (explore)**: 發現新房間
- **結束回合 (endTurn)**: 結束當前回合

### 3. 決策評分系統

每個決策都會根據以下因素評分：

#### 移動決策
- 基礎分數: 10
- 朝向敵人移動: +20 × attackWeight
- 危急時遠離敵人: +30 × survivalWeight

#### 攻擊決策
- 基礎分數: 15
- 力量優勢: +5 × 優勢值
- 攻擊虛弱目標: +25
- 危急狀態懲罰: -20

#### 使用物品決策
- 基礎分數: 12
- 戰鬥前使用武器: +20
- 危急時使用: +15

#### 探索決策
- 基礎分數: 8
- 探索階段加成: +15
- 物品少時加成: +10

### 4. 難度等級

#### Easy (簡單)
- 攻擊權重: 0.5
- 目標權重: 0.8
- 生存權重: 0.6
- 50% 機率選擇次優決策

#### Medium (中等)
- 攻擊權重: 1.0
- 目標權重: 1.2
- 生存權重: 1.0
- 20% 機率選擇次佳決策

#### Hard (困難)
- 攻擊權重: 1.5
- 目標權重: 2.0
- 生存權重: 1.3
- 總是選擇最佳決策

### 5. 叛徒行為模式

#### 作祟階段
1. **優先攻擊**: 當英雄在同一房間時，優先攻擊
2. **追擊英雄**: 朝向最近英雄移動
3. **使用武器**: 戰鬥前使用可用武器
4. **避免死亡**: 危急時考慮逃跑

#### 探索階段
1. **探索新房間**: 發現新房間以獲取資源
2. **收集物品**: 尋找有用的物品和預兆

## 假設與限制

### 假設
1. AI 只能使用 Rules Engine 提供的合法行動
2. AI 不會作弊或獲取隱藏資訊
3. AI 決策基於當前可見的遊戲狀態

### 限制
1. 目前不支援複雜的長期規劃
2. 不會預測對手行動
3. 某些特殊作祟規則需要手動實作

## 確定性與重播

AI 使用有種子的隨機數生成器，確保：
- 相同種子產生相同的決策序列
- 支援遊戲重播功能
- 便於除錯和測試

## 未來改進

1. **強化學習**: 使用機器學習改進決策品質
2. **長期規劃**: 實作 A* 或蒙特卡洛樹搜尋
3. **情境感知**: 根據不同作祟劇本調整策略
4. **協作 AI**: 支援多個叛徒的協作行為

## 使用方式

```typescript
import { createTraitorAI } from '@betrayal/game-engine/ai';

// 建立 AI
const ai = createTraitorAI('player-id', 'hard', 'seed-123');

// 執行回合
const decisions = ai.executeTurn(gameState);

// 取得決策
const decision = ai.decideMove(gameState);
```
