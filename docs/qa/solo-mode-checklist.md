## 使用說明

本檢查清單用於快速驗證單人模式功能，可在開發過程中或發布前執行。每個檢查項目應標記：

- [ ] 未測試
- [x] 通過
- [-] 失敗 (請註明缺陷編號)
- [~] 不適用
